Data Cleaning & Reporting Automation

Project Overview
This project automates the process of cleaning raw sales data and generating reports using Python, Excel, and Power BI.

The system removes duplicate records, handles missing values, standardizes inconsistent data, and creates visual summaries for business insights.


# Features

- Automated data cleaning
- Remove duplicate records
- Handle missing values
- Standardize text formatting
- Generate cleaned Excel reports
- Create sales summary charts
- Power BI dashboard visualization
- Automated reporting workflow


# Technologies Used

- Python
- Pandas
- Matplotlib
- Excel
- Power BI
